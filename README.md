# 🚀 Quiz Application (Frontend Project)

## 📌 Overview
A responsive and interactive Quiz Application developed using HTML, CSS, and JavaScript. The application allows users to attempt multiple-choice questions, track their scores, and store high scores using browser local storage.

This project demonstrates strong fundamentals in frontend development, problem-solving, and user interaction design, making it suitable for placements and internships.

---

## 🎯 Key Highlights
- Built a complete quiz system without using any frameworks  
- Implemented dynamic question rendering using JSON data  
- Used Local Storage API to store and retrieve high scores  
- Designed a multi-page flow (Start → Quiz → End → High Scores)  
- Focused on clean UI and smooth user experience  

---

## 🛠️ Tech Stack
- HTML5 – Structure  
- CSS3 – Styling and responsiveness  
- JavaScript (ES6) – Logic, DOM manipulation, and interactivity
